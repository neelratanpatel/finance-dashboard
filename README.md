# 📈 Professional Finance Dashboard

A clean, responsive, and data-driven finance dashboard built with **React**, **Redux Toolkit**, and **Recharts**.

## 🚀 Features
- **Global State Management:** Powered by Redux for real-time updates across all components.
- **Dynamic Charts:** Visualizes cumulative balance trends, income, and expenses with custom tooltips.
- **Role-Based UI:** Switch between **Admin** (Edit/Delete access) and **Viewer** (Read-only) modes.
- **Fully Responsive:** Optimized for Mobile (iPhone SE), Tablets, and Desktop using Custom CSS (no utility frameworks).
- **Localization:** Formatted for Indian Rupee (₹) and `en-IN` numbering systems.

## 🛠️ Tech Stack
- **Frontend:** React 19 (Vite)
- **State:** Redux Toolkit
- **Icons:** Lucide-React
- **Charts:** Recharts
- **Styling:** Modular Custom CSS (Mobile-first approach)

## 📦 Setup Instructions
1. Clone the repo: `https://github.com/neelratanpatel/finance-dashboard.git`
2. Install dependencies: `npm install`
3. Run local server: `npm run dev`
